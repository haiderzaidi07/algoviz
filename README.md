# AlgoViz

An application to visualize performance of Kruskal's and Primâ€™s algorithms on graphs, either sparse and dense, with inputs ranging from 5 to 250 nodes

## Live Demo

The live demo of this website can be accessed at: https://algovyz.netlify.app/


## Features

- Custom graph size
- Ability to drag and move nodes around
- Measures performance of algorithms upto 0.1 milliseconds


## Tech Stack

**Client:** TypeScript, React, TailwindCSS, D3.js


## Run Locally

Clone the project

```bash
git clone https://github.com/haiderzaidi07/algoviz.git
```

Go to the project directory

```bash
cd algoviz/
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run dev
```
